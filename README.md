# ALDS cpp

Algorithm and Data Structure implemented in C++ 20 for learning purpose.

## Requirements

- CMake 3.22 or later
- C++ 20 compatible compiler
- Google Benchmark (optional)

## Usage

### How to build

```bash
$ mkdir build
$ cd build
$ cmake .. -DCMAKE_BUILD_TYPE=Release
$ make
```

### Run tests

```bash
$ ctest
```

### Run benchmarks

```bash
$ ./benchmarks/alds_bench
```
